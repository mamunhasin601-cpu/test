# Wildberries — требования к фото и видео карточки

**Источники:** справочный центр WB (`seller.wildberries.ru/instructions`), sellermoon.ru, postium.ru, sellerart.ru, trustyone.ru. Данные на 2026 год.

## Фото — технические параметры

| Параметр | Значение |
|---|---|
| Рекомендуемый размер | **900×1200 px** (оптимально 1200×1600 px) |
| Минимальный размер | 700×900 px |
| Максимальный размер | до 8000 px по одной стороне |
| Соотношение сторон | **строго 3:4 (вертикальное)** |
| Максимальный вес | до 10 МБ (реально лучше 1–3 МБ) |
| Форматы | JPG, PNG, WebP |
| Цветовой профиль | sRGB IEC 61966-2.1 (не CMYK, не Adobe RGB) |
| DPI | 72 |
| Кол-во фото | до **30** на карточку |
| Видео | 1 файл (+ возможность «Фото 360») |

## Главное фото — правила

- Товар занимает **70–85 %** площади кадра (для обуви/аксессуаров минимум 10 %, но лучше больше).
- Фон: белый или светлый нейтральный, контрастный товару. Разрешён интерьерный/яркий монохром для еды и косметики.
- Точное соответствие реальному товару (цвет, комплект, модель).
- **Разрешён текст/заголовок на карточке** (в отличие от Ozon и YM) — WB допускает инфографику и на главном слайде для конкурентных ниш, но злоупотребление снижает CTR.

### Слепые зоны интерфейса WB

WB накладывает свои элементы поверх фото — важные детали держать в центре, отступ от каждого края **10–15 %** ширины кадра.

| Зона | Что накладывается |
|---|---|
| Нижний левый угол | плашка со скидкой (%) |
| Верхний правый угол | «сердечко» (в избранное) |
| Верхний левый угол | системные бейджи «Хит», «New», «Eco» |
| Нижний правый угол | кнопка быстрого добавления в корзину |

Плашки маркетплейса могут перекрывать до **15 %** изображения — не размещать в этих зонах ключевой текст и логотип.

## Дополнительные фото

- Единый стиль (фон, цветокоррекция, композиция) — прямое требование WB.
- Рекомендованный набор: 5–8 качественных слайдов (не «максимум ради количества»).
- Разрешена инфографика: текст, иконки, графика на цветном фоне.
- Логика воронки: общий вид → детали → в использовании / на модели → размеры → комплектация → размерная сетка → преимущества → состав.

## Что запрещено на фото

- логотипы сторонних брендов, водяные знаки;
- контакты продавца (телефон, email, соцсети, мессенджеры), QR-коды, ссылки;
- упоминание других маркетплейсов;
- слово «скидка» без согласования с площадкой;
- агрессивные призывы к покупке, обещания вознаграждения за отзыв;
- элементы, имитирующие интерфейс маркетплейса (фейковые кнопки/бейджи);
- недостоверное изображение товара, коллажи не того комплекта.

## Видео — технические параметры

| Параметр | Значение |
|---|---|
| Формат | **MP4, MOV** (кодек H.264), также FLV |
| Соотношение сторон | **3:4** (строго) |
| Рекомендуемое разрешение | 900×1200 px или 1080×1440 px (также встречается требование 720×1920) |
| Длительность | 10–60 сек (оптимум 15–20 сек), максимум 3 минуты |
| Вес файла | до **50 МБ** |
| Нижние 15 % высоты | перекрываются плеером — важное держать в центре |

### Фото 360°

- 12–36 кадров (оптимум ~24);
- размер как у фото (например, 900×1200 px);
- камера зафиксирована, объект строго по центру.

## Частые причины отклонения модерацией

1. Превышение веса файла (>10 МБ).
2. Несоответствие пропорций 3:4.
3. Низкое разрешение / размытие / шумы.
4. Пёстрый или неподходящий фон.
5. Логотипы или водяные знаки.
6. CMYK вместо sRGB (искажение цвета).
7. Мелкий объект («эффект спичечной коробки») — товар <70 % кадра.
