## Ключевой вывод для нашего генератора

Универсальный «безопасный» пресет:
- **Размер:** 1200×1600 px (пропорция 3:4), плюс отдельная версия 1024×1024 (1:1) под YM.
- **Фон:** белый `#FFFFFF` (для Ozon и YM — обязателен, для WB — рекомендован).
- **Товар в кадре:** 70–85% площади, отступ 8–12% от каждого края (безопасная зона под плашки WB/Ozon).
- **Текст:** на главном фото — запрещён (Ozon/YM/WB — модерация). Инфографика — только со 2-го слайда.
- **Формат файла:** JPG качества ≥85% (фото) / PNG (инфографика), цвет sRGB, вес ≤8 МБ.
- **Видео:** MP4/H.264, 9:16 (720×1280 / 900×1200), 15–30 сек, вес ≤50 МБ, без звука/со звуком + субтитры.
